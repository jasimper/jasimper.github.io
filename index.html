Some of My Work

Salty Paws is a website created for volunteers who deditcate their time to caring for free-roaming cats. This site, which I co-created with 3 awesome classmates, was my final project for The Iron Yard.

Skills/Tools used: Rails, PostgreSQL, HTML5, CSS3/Sass, React, Git/GitHub, Pivotal Tracker, Minitest

Hex Mex is created by parsing a JSON file of titles, keywords, and hex values for multiple color palletes. With JavaScript and the DOM API, the palletes are rendered onto a webpage, then styled with CSS.

Skills used: JavaScipt, DOM API, CSS3, HTML, Git/Github

Darth Vader's To-do List is another fun exercise in JavaScript. By just querying the DOM and building HTML strings with JavaScript, then posting them to the DOM, items are added to the to-do list where they can be marked as complete and deleted.

Skills used: JavaScript, CSS3, HTML, Git/Github
